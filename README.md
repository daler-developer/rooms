# Room Chat App

## Features

- Realtime messages using WebSocket
- Schedule messages
- User online status
- User typing status
- Invitations
- Custom form library
- Custom ui library

## Tech stack:

- React
- GraphQL
- TypeScript
- PostgreSQL
- Node.js
- Superbase
